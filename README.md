# Psychotherapist Template

![Psychotherapist template cover screenshot](./public/project-cover.png 'Psychotherapist template in desktop version')

Inspiring website template for a psychotherapist or psychologist. Design structured with 6 pages: *Home*, *About*, *Contact*, *Help*, *Testimonials*, and *Error* page. Contact form functionality with **Web3Forms**.

### Developed with:

- Astro Framework
- TypeScript
- Bootstrap
- SASS
- Google Maps
- Web3Forms

## 🧞 Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |
| `npm run styles`          | Compile .sass and .scss files to .css files      |
|                           |                                                  |
